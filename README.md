## Status
- Prototype

## Features
- Central joystick for mouse-like control
- Additional small tactile switches for extended input
- Ergonomic ~5° inward tilt design
- Hybrid input (keyboard + pointing device)
- Case made from thick paper and leather


## Specifications

| Item | Description |
|-----|-----|
| Layout | freestyle |
| Keys | 52 keys |
| Controller | RP2040-Zero |
| Firmware | QMK |
| Connection | USB-C |
